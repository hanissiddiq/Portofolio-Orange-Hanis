## 📖 About

Hanis Portfolio merupakan website portofolio pribadi yang dibangun menggunakan teknologi modern untuk menampilkan profil profesional, pengalaman kerja, riwayat proyek, kemampuan teknis (Tech Stack), sertifikat, serta informasi kontak.

Website ini memiliki tampilan yang modern, responsif, ringan, dan dioptimalkan untuk memberikan pengalaman pengguna yang baik di berbagai perangkat.

# 📋 Requirements

Pastikan perangkat telah terinstal:

- Node.js 20.x atau lebih baru
- npm versi terbaru

Cek versi:

```bash
node -v
npm -v
```

---

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/username/hanis-portfolio.git
```

Masuk ke folder project

```bash
cd hanis-portfolio
```

Install seluruh dependency

```bash
npm install
```

---

# ▶️ Running Development Server

Jalankan project

```bash
npm run dev
```

Project akan berjalan pada

```
http://localhost:3000
```

Karena project menggunakan konfigurasi Vite:

```json
vite --port=3000 --host=0.0.0.0
```

maka website dapat diakses dari browser maupun perangkat lain dalam jaringan yang sama.

---

# 🏗 Build Production

Untuk membuat build production

```bash
npm run build
```

Hasil build akan berada pada folder

```
dist/
```

---

# 👀 Preview Production

Untuk menjalankan hasil build
